# Testing Plan

## Testing Goal

The project should be tested like a real project, but not every visual detail needs an automated test. Focus on server logic and important flows.

## Backend Tests

Use JUnit and Mockito where possible.

### JwtServiceTest

Test:

- Creates valid JWT.
- Extracts username/userId/role.
- Rejects invalid token.
- Rejects expired token.

### CookieUtilsTest

Test:

- Creates auth Cookie.
- Cookie has correct name.
- Cookie has correct maxAge.
- Clears Cookie on logout.
- Extracts Cookie value from request.

### RoomCodeServiceTest

Test:

- Generates non-empty code.
- Code has expected length.
- Code uses allowed characters.
- Avoids duplicate code when repository says code exists.

### SubjectServiceTest

Test:

- Returns only active subjects.
- Finds subject by id.
- Throws NotFoundException for missing subject.

### RaceServiceTest

Test:

- Creates race for current teacher.
- Validates maxPlayers <= 8.
- Validates subject exists.
- Generates roomCode.
- Teacher can only access their own races.

### TeacherDashboardServiceTest

Test:

- Returns teacher name.
- Returns only teacher races.
- Calculates total/active/finished race counts.

## Controller Tests

Add later if there is enough time:

- `AuthControllerTest`
- `TeacherRaceControllerTest`
- `TeacherDashboardControllerTest`

Test with MockMvc when possible.

## Frontend Manual Tests for Stage A

### Login

1. Open `/login`.
2. Enter valid teacher credentials.
3. Submit.
4. Confirm redirect to `/teacher/dashboard`.
5. Confirm Cookie exists in browser devtools.

### Refresh Session

1. Login.
2. Refresh page.
3. Confirm user remains logged in.
4. Confirm `/api/auth/me` was called.

### Protected Route

1. Clear Cookie.
2. Try to open `/teacher/dashboard`.
3. Confirm redirect to `/login`.

### Create Race

1. Login as teacher.
2. Open dashboard.
3. Click Create Race.
4. Choose Subject from server.
5. Enter title, max players, total distance.
6. Submit.
7. Confirm race appears in list.
8. Confirm roomCode appears.

### Race Room

1. Click a race from dashboard.
2. Confirm route opens.
3. Confirm page shows title, subject, room code, status, current players.

### Logout

1. Click logout.
2. Confirm Cookie is removed.
3. Confirm redirect to login.
4. Confirm dashboard is blocked.
